# Churrascômetro


<p>Simples projeto que tem como funcionalidade mostrar a quantidade de carne em kg, cerveja em latas e bebida em litros para um churrasco de média ou longa duração.</p>

<h2>Linguagens usadas</h2>
<ul>
  <li>JavaScript</li>
  <li>HTML</li>
  <li>CSS</li>
</ul>
